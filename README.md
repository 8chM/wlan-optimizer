# WLAN-Optimizer

Open-Source-Tool zur Planung, Messung und Optimierung von WLAN-Infrastrukturen in Heimnetzwerken.

## Funktionen

- **Grundrissplanung** - Grundriss hochladen, Wände einzeichnen, Access Points platzieren
- **Heatmap-Berechnung** - Konservative Abdeckungsprognosen für 2,4 GHz und 5 GHz
- **Optimierungsassistent** - Geführte Messläufe mit realen Daten
- **Mixing Console** - Interaktive Anpassung von AP-Parametern mit Live-Vorschau
- **Berichte** - Vorher-/Nachher-Vergleich und Export

## Status

Projekt befindet sich in der Planungsphase.
